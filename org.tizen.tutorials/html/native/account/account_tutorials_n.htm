Account: Managing Account Data

The account tutorials demonstrate how to use the following features in creating Tizen native applications.

  • Account Manager: Managing Account Information on the Device

    Demonstrates how you can connect to and disconnect from the Account Service, create, delete, and update accounts in the account database, and manage account information.

  • OAuth 2.0: Using the Authorization Framework

    Demonstrates how you can obtain access tokens using OAuth 2.0 authorization.

  • Sync Manager: Scheduling for Application Data Syncing

    Demonstrates how you can receive notifications about syncing operations between a server and the device.

  • FIDO Client: Authenticating users using FIDO Framework

    Demonstrates how you can authenticate users using the FIDO framework.
