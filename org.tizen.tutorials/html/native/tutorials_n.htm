Tizen
Native Tutorials

Tizen native tutorials teach you how to incorporate Tizen features into your application. The tutorials cover detailed code snippets to help you see how to write the required code in practice.

Select the API domain you are interested in and see how you can implement its various functionalities into your application:

  • Application Framework: Controlling Your Application

    Demonstrates how you can configure application properties and manage application data.

  • Base: Using Fundamental Libraries

    Demonstrates how you can manage locale-sensitive information and use SQL databases in your application.

  • Content: Getting and Handling Media Content

    Demonstrates how you can manage and store a variety of data types, input and output data, and device content.

  • Context: Detecting and Exploiting Contextual Information

    Demonstrates how you can detect user activities and gestures.

  • Graphics: Creating and Managing Graphics

    Demonstrates how you can create and edit various kinds of images, shapes, text, and elements, and recognize specific elements from images and video.

  • Location: Using Location Information and Service

    Demonstrates how you can manage location data.

  • Messaging: Sending SMS, MMS, Email, and Push Messages

    Demonstrates how you can send and receive SMS, MMS, email, and push messages.

  • Multimedia: Handling Audio, Video, Camera, and Images

    Demonstrates how you can capture and play various media.

  • Network: Managing Connections and Communication

    Demonstrates how you can send data to networks and other applications, and receive data back from them.

  • OAuth2: Using the Authorization Framework

    Demonstrates how you can obtain access tokens using oauth2 authorization.

  • Security: Handling Keys and Cryptographic Operations

    Demonstrates how you can increase the security of your application through cryptography and request control.

  • Social: Managing Personal Data

    Demonstrates how you can manage the user's personal data, such as contacts, calendars, and call history, on the device.

  • System: Getting System Information and Managing Devices

    Demonstrates how you can manage data related to or provided by the device.

  • Telephony: Handling Phone Call, SIM Card, and Network Information

    Demonstrates how you can retrieve call status, extract information stored on a SIM card, and get information about the current cellular network and telephony service.

  • UI Framework: Creating the Application UI

    Demonstrates how you can create a visual outlook for your application to ensure the best possible user experience.

  • UIX: Taking Advantage of UI Extensions

    Demonstrates how you can recognize and synthesize speech.

  • Web: Managing Web Pages and Web Content

    Demonstrates how you can create a simple web browser

  • Sync: Scheduling for Application Data Syncing in mobile applications only

    Demonstrates how you can receive notifications about syncing operations between a server and the device.
