UI Framework: Creating the Application UI

The UI framework tutorials demonstrate how to create and manage the UI and its various elements and events in Tizen native applications:

  • EFL: Using EFL Features for Designing the UI

    Demonstrates how you can use EFL to create your application UI.

  • Evas: Creating and Managing Image Objects

    Demonstrates how you can create and manage image objects.

  • Event and Effect: Using Animations and Effects

    Demonstrates how you can create and implement a variety of animation effects and use gestures in applications.

  • UI Components: Managing UI Elements

    Demonstrates how you can manage user interface-related features.

  • Efl Extension: Creating a Naviframe Window

    Demonstrates how you can register the EFL Extension Menu and Back key events.

  • Font Setting: Using Various Fonts in the Application

    Demonstrates how you can modify the fonts and text size of your application, or set your own font.

  • EFL UTIL: Adjusting the Notification Window Level

    Demonstrates how you can create a notification window and adjust its position.

  • EFL Optimization: Optimizing Tizen Native Applications

    Demonstrates how you can optimize your Tizen native application.

  • EOM: Using External Output Devices

    Demonstrates how you can use external output devices.
