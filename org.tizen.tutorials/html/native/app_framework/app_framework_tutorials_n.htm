Application Framework: Controlling Your Application

The application framework tutorials demonstrate how to use the following features in creating Tizen native applications:

  • Application: Using Alarms, Preferences, AppControl, and Other Fundamentals

    Demonstrates how you can manage application fundamentals, such as application settings, alarms, application controls, and application groups.

  • Application Manager: Getting Information about Applications

    Demonstrates how you can get information about applications.

  • Badge: Displaying the Notification (Badge) Count on the Home Screen

    Demonstrates how you can create and modify badges.

  • Bundle: Using the String-based Dictionary ADT

    Demonstrates how you can create and manage a key-value dictionary.

  • Data Control: Exchanging Specific Data Between Applications

    Demonstrates how you can manage map data and SQL data.

  • Message Port: Passing Messages Between Applications

    Demonstrates how you can send and receive messages through message ports.

  • Notification: Notifying the User of Application Events

    Demonstrates how you can display notifications on the application screen.

  • Package Manager: Installing and Uninstalling Applications

    Demonstrates how you can manage and retrieve package information.

  • Service Application: Creating Background Applications

    Demonstrates how you can create and manage native service applications.

  • Shortcut: Adding a Shortcut to Home

    Demonstrates how you can create and modify shortcuts and widgets on the home screen.

  • Widget Application: Creating Widget Applications

    Demonstrates how you can create and manage widget applications.

The following tutorials apply in wearable applications only:

  • Watch Application: Creating a Watch Application

    Demonstrates how you can create a watch application, set the time, and use the ambient mode.
