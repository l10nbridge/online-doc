Context: Detecting and Exploiting Contextual Information

The context tutorials demonstrate how to use the following features in creating Tizen mobile native applications:

  • Activity Recognition: Detecting User Activities

    Demonstrates how you can detect user activities.

  • Gesture Recognition: Detecting User Gestures

    Demonstrates how you can detect gestures performed with the device.

  • Contextual Trigger: Context-aware Task Automation

    Demonstrates how you can automate your tasks, by monitoring contextual events and conditions detected on the device.

  • Contextual Trigger: Getting the Device Usage Profiles

    Demonstrates how you can retrieve the device usage profiles analyzed from the usage history of the device.
