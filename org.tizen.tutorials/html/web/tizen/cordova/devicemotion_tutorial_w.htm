Device Motion: using accelerometer motion sensor

This tutorial demonstrates how you can get access to the device's accelerometer. The navigator.accelerometer object allows access to the functionality of the DeviceMotion API.

Get Current Acceleration

Get the current acceleration along the x, y, and z axes.

  1. Prepare callback method to be invoked with current acceleration values

    function onSuccess(acceleration) {
  console.log('Acceleration X: ' + acceleration.x + '\n' +
              'Acceleration Y: ' + acceleration.y + '\n' +
              'Acceleration Z: ' + acceleration.z + '\n' +
              'Timestamp: '      + acceleration.timestamp);
}

  2. Prepare callback method called when errors occur

    function onError() {
  console.log('onError!');
}

  3. Call getCurrentAcceleration method and pass your callbacks

    navigator.accelerometer.getCurrentAcceleration(onSuccess, onError);

Watch Acceleration Values

You can get acceleration along the x, y, and z axes at a regular interval.

  1. Prepare callback method to be invoked with current acceleration values. 

    function onSuccess(acceleration) {
  console.log('Acceleration X: ' + acceleration.x + '\n' +
              'Acceleration Y: ' + acceleration.y + '\n' +
              'Acceleration Z: ' + acceleration.z + '\n' +
              'Timestamp: '      + acceleration.timestamp);
  console.log('Please wait 3 seconds for the next measurement...');
}

  2. Prepare callback method called when errors occur

    function onError() {
  console.log('onError!');
}

  3. Specify interval of calling onSuccess in miliseconds.

    var options = { frequency: 3000 };  // Update every 3 seconds

  4. Call watchAcceleration with options

    var watchID = navigator.accelerometer.watchAcceleration(onSuccess, onError, options);

  5. Regular acceleration measurments now are taking place, and each time a callback is called.

  6. When finished, stop watching the acceleration value by calling clearWatch with watchID as an argument. 

    navigator.accelerometer.clearWatch(watchID);
