Slider

Slider is a draggable UI component that is used to select a value within a certain range. This documentation explains the basic usages and features of sliding component. For more information, see the API Reference

Basic Usage

  1. Make a new slider as a child of the parent 
    Evas_Object *slider = NULL ;

slider = elm_slider_add(parent);
  2. Configure the slider. The following configurations are optional. Unless you set what you need, a slider adopt default styles.
    • Set styles of the slider. 
      // Default : "default"
elm_object_style_set(slider,"default");
    • Set another options. For more information, see the various slider practices. 
      elm_slider_horizontal_set(slider, EINA_FALSE);
elm_slider_inverted_set(slider, EINA_FALSE);
elm_slider_indicator_show_set(slider, EINA_TRUE);
elm_slider_indicator_format_set(slider, "%.1f");
  3. Deal with event.
    • Register callback functions 
      evas_object_smart_callback_add(slider, "changed", slider_changed_cb, 0);
    • Implement callback function 
      static void
slider_changed_cb(void *data, Evas_Object *obj, void *event_info)
{
	double v = elm_slider_value_get(obj);
	dlog_print(DLOG_INFO, LOG_TAG, "The slider value has changed : %0.1f\n", v);
}

The following example shows a simple use case of slider component:

Example: A simple use case of slider component 
Evas_Object *box, *slider, *nf ;

// Starting right after the basic EFL UI layout code
// (win - conformant - naviframe)

box= elm_box_add(nf);
elm_naviframe_item_push(nf, "Slider", NULL, NULL, box, NULL);

// Add a slider
slider = elm_slider_add(box);
evas_object_size_hint_align_set(slider, EVAS_HINT_FILL,0.5);

// Set a style whatever you want
elm_object_style_set(slider,"center_point");

// Set a range of the slider
elm_slider_min_max_set(slider, 0.0, 10.0);
elm_slider_value_set(slider, 5.0);

evas_object_show(slider);
elm_box_pack_end(box, slider);

Various Slider Practices

You could build various sliders by combining styles and options.

  1. elm_slider_indicator_show_set(slider_object, Eina_Bool)
    Example: Slider indicator
    Example code EINA_TRUE EINA_FALSE 
    Evas_Object *slider, *nf, *box;

// Add a box to contain a slider
box= elm_box_add(nf);
elm_naviframe_item_push(nf, "Slider", NULL, NULL, box, NULL);

// Add a slider
slider = elm_slider_add(box);
evas_object_size_hint_align_set(slider, EVAS_HINT_FILL,0.5);

// Set styles in various ways
elm_object_style_set(slider,"default");
elm_slider_indicator_show_set(slider, EINA_TRUE);
elm_slider_indicator_format_set(slider, "%.2f");

// Set a range of the slider
elm_slider_min_max_set(slider, 0.0, 10.0);
elm_slider_value_set(slider, 5.0);

evas_object_show(slider);
elm_box_pack_end(box, slider);
  2. elm_slider_horizontal_set(slider_object, Eina_Bool)
    Example: Slider orientation
    Example code EINA_TRUE EINA_FALSE 
    Evas_Object *slider, *nf, *box;

box= elm_box_add(nf);
elm_naviframe_item_push(nf, "Slider", NULL, NULL, box, NULL);

// Add a slider
slider = elm_slider_add(box);
evas_object_size_hint_align_set(slider,EVAS_HINT_FILL,EVAS_HINT_FILL);

// Set a orientation of the slider

evas_object_show(slider);

Callbacks

You can register callback functions connected to the following signals for a slider object:

Table: Slider callback signals
Signal Description event_info
changed Whenever the slider value is changed by the user. NULL
delay,changed A short time after the value is changed by the user. This will be called only when the user stops dragging for a very short period or when they release their finger/mouse, so it avoids possibly expensive reactions to the value change. NULL
slider,drag,start Dragging the slider indicator around has started. NULL
slider,drag,stop Dragging the slider indicator around has stopped. NULL

The signals in API Reference might be more extensive, but only the signals stated above are actually supported by Tizen.
