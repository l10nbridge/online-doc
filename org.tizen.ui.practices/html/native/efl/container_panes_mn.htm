Panes

Panes is a container component that adds a draggable bar between 2 sections of content. The sections are resized when the bar is dragged. For more information, see API Reference.

Basic Usage

Take the following steps to use a panes in your application:

  1. Add a panes with elm_panes_add() function. 
    Evas_Object *panes;

panes = elm_panes_add(parent);
  2. Add content to the panes with elm_object_part_content_set() function. You can add a content to the left pane. 
    Evas_Object *left_button;

elm_object_part_content_set(panes, "left", left_button);
  3. You can add a content to the right pane. 
    Evas_Object *right_button;

elm_object_part_content_set(panes, "right", right_button);
  4. Set the direction of the panes with elm_panes_horizontal_set() function. A panes is vertical by default. Horizontal panes have "top" and "bottom" contents, vertical panes have "left" and "right" contents. 
    elm_panes_horizontal_set(panes, EINA_TRUE);
elm_object_part_content_set(panes, "top", top_button);
elm_object_part_content_set(panes, "bottom", bottom_button);
  5. Set the size of the panes with elm_panes_content_right_size_set function. By default both sides of panes have the same size. You can change the size proportion of panes. If the left content should be displayed over 75% of the panes size, The size should be passed as 0.75. This way, right content will be resized to 25% of panes size. If displayed vertically, left content is displayed at top, and right content at bottom 
    elm_panes_content_left_size_set(panes, 0.75);
  6. Register callback functions. 
    evas_object_smart_callback_add(panes, "clicked,double", clicked_double_cb, data);

    void clicked_double_cb(void *data, Evas_Object *obj, void *event_info)
{
   dlog_print(DLOG_INFO, LOG_TAG, "Panes double clicked\n");
}

The following example shows a simple use case of panes container:

Example: A simple use case of panes container 
Evas_Object *win, *conf, *nf, *panes, *left_button, *right_button;

// Starting right after the basic EFL UI layout code
// (win - conformant - naviframe)

// Add a panes to contain two buttons and push the panes into the naviframe
panes = elm_panes_add(nf);
elm_object_content_set(nf, panes);
evas_object_show(panes);
elm_naviframe_item_push(nf, "Panes", NULL, NULL, panes, NULL);

// Add a button to the left pane
left_button = elm_button_add(panes);
elm_object_text_set(left_button, "Left");
elm_object_part_content_set(panes, "left", left_button);
evas_object_show(left_button);

// Add a button to the right pane
right_button = elm_button_add(panes);
elm_object_text_set(right_button, "Right");
elm_object_part_content_set(panes, "right", right_button);
evas_object_show(right_button);

Callbacks

You can register callback functions connected to the following signals for a panes object:

Table: Panes callback signals
Signal Description event_info
clicked This callback function registered for user operation is called whenever the panes is clicked(press/release). NULL
clicked,double This callback function registered for user operation is called whenever the panes is double clicked. NULL
pressed This callback function registered for user operation is called whenever the panes is pressed. NULL
unpressed This callback function registered for user operation is called whenever the panes is released after being pressed. NULL

The signals in API Reference might be more extensive, but only the signals stated above are actually supported by Tizen.
