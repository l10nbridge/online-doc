Radio

This feature is supported in wearable applications only.

Radio is a UI component which displays one or more options and allows users to select one of them. This document teaches you the basic usage and main features of radio component. For more information, see API Reference.

Basic Usage

Take the following steps to use radio buttons:

  1. Add as many radio buttons as you want with elm_radio_add() function. 
    Evas_Object *radio;
radio = elm_radio_add(parent);
  2. Set a value and then group radio buttons.
    • Assign a unique value to each radio button with elm_radio_state_value_set() function. 
      elm_radio_state_value_set(radio, 1);
    • Group radio buttons with elm_radio_group_add() function. You can set one of the radio buttons as selected with elm_radio_value_set() function. 
      elm_radio_group_add(radio, group);
    • Group the radio buttons visually by packing into one box.
  3. Register callback functions. 
    evas_object_smart_callback_add(radio, "changed", changed_cb, data);

    void
changed_cb(void *data, Evas_Object *obj, void *event_info)
{
	 dlog_print(DLOG_INFO, LOG_TAG, "The value has changed\n");
}
Example: A radio 
Evas_Object *nf, *box, *radio, *group;

// Starting right after the basic EFL UI Layout code
// win - conformant - naviframe

box = elm_box_add(nf);
evas_object_show(box);
elm_naviframe_item_push(nf, "Radio", NULL, NULL, box, NULL);

// Radio 1
// 1. add a radio
radio = elm_radio_add(box);
// 2. configure the radio
elm_radio_state_value_set(radio, 1);
evas_object_show(radio);
elm_box_pack_end(box, radio);

group = radio;

// Radio 2
radio = elm_radio_add(box);
elm_radio_state_value_set(radio, 2);
evas_object_show(radio);
elm_box_pack_end(box, radio);
// add
elm_radio_group_add(radio, group);

// Radio 3
radio = elm_radio_add(box);
elm_radio_state_value_set(radio, 3);
evas_object_show(radio);
elm_box_pack_end(box, radio);
elm_radio_group_add(radio, group);

// set one of the radio components as selected
elm_radio_value_set(group, 1);

Callbacks

You can register callback functions connected to the following signals for a radio object:

Table: Radio callback signals
Callback Description event_info
changed The radio button is selected. NULL

The signals list in the AR is more extensive, but only the signals in this topics are actually supported by Tizen.
