Progressbar

This feature is supported in wearable applications only.

The progressbar component is used to display the progress status of a given job. It inherits from the layout component, which means that layout functions can be used on the progressbar component.

For more information, see the Progress bar API.

Figure: Progressbar hierarchy

Adding a Progressbar Component

To create a progressbar component, use the elm_progressbar_add() function:

Evas_Object *pb = elm_progressbar_add(win);

Configuring the Progressbar

To set the progressbar styles:

  • The following styles are available for a rectangular UI screen:

    • default
    • pending_list
    • process
    • process/groupindex
    • process/popup/small

  • The following styles are available for a circular UI screen:

    • process
    • process/small
elm_object_style_set(pb, "pending_list");

To configure the progressbar for the rectangular UI screen:

  • Activate the progressbar pulse mode to make the progressbar loop infinitely between the start and end position:

    elm_progressbar_pulse_set(pb, EINA_TRUE);
elm_progressbar_pulse(pb, EINA_TRUE);

  • Invert the progressbar.

    In the inverted mode, the high values are on the left and the low values on the right.

    elm_progressbar_inverted_set(pb, EINA_TRUE);

Using the Progressbar

To use the progressbar:

  • Read the current value with the elm_progressbar_value_get() function:

    double value = elm_progressbar_value_get(pb);

  • Change the value with the elm_progressbar_value_set() function.

    To set the value to 20%:

    elm_progressbar_value_set(pb, 0.2);

Using the Progressbar Callbacks

To receive notifications about the progressbar events, listen to the changed signal, which is called when the progressbar value changes.

Note
The signal list in the API reference can be more extensive, but only the above signal is actually supported in Tizen.

To register and define a callback for the changed signal:

{
   evas_object_smart_callback_add(pb, "changed", changed_cb, data);
}

// Callback for the "changed" signal
// Called when the progressbar value changes
void 
changed_cb(void *data, Evas_Object *obj, void *event_info)
{
   dlog_print(DLOG_INFO, LOG_TAG, "The value has changed\n");
}
