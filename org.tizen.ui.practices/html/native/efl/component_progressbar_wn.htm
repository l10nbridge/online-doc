Progressbar

Progressbar is a UI component which can display the progress status of a given job. For more information, see API Reference.

Basic Usage

Take the following steps to use a progressbar:

  1. Add a progressbar with elm_progressbar_add() function. 
    2. Evas_Object *progressbar;

progressbar = elm_progressbar_add(parent);
  2. Set a style are available for a rectangular or circular UI screen
    • Set a style to the progressbar with elm_object_style_set() function. You can use the default style without this step.
      • elm_object_style_set(progressbar, "pending_list");
  3. Activate the progressbar.
    • Set a the progressbar pulse mode to make activate progressbar with elm_progressbar_pulse_set() function: 
      • elm_progressbar_pulse_set(progressbar, EINA_TRUE);
    • Use elm_progressbar_pulse()to start loop the progressbar animation:
      • elm_progressbar_pulse(progressbar, EINA_TRUE);
  4. Register callback functions.
    5. evas_object_smart_callback_add(progressbar, "changed", changed_cb, data);
    void changed_cb(void *data, Evas_Object *obj, void *event_info)
{
   dlog_print(DLOG_INFO, LOG_TAG, "The value has changed\n");
}

The following example shows a simple use case of progressbar component:

Example: A simple use case of Progressbar component in rectangular and circular UI screen 
Evas_Object *win, *conf, *nf, *box, *progressbar;

// Starting right after the basic EFL UI layout code
// (win - conformant - naviframe)

// Add a box to contain a progressbar and push the box into the naviframe
box = elm_box_add(nf);
evas_object_show(box);
elm_naviframe_item_push(nf, "Progressbar", NULL, NULL, box, NULL);

// Add a progressbar and set a style "default"
progressbar = elm_progressbar_add(box);

// Set a progressbar alignment. The progressbar fills whole parent area
evas_object_size_hint_align_set(progressbar, EVAS_HINT_FILL, EVAS_HINT_FILL);
// Set a progressbar weight size. The progessbar takes up all the space in its parent
evas_object_size_hint_weight_set(progressbar, EVAS_HINT_EXPAND, EVAS_HINT_EXPAND);

// Set a progressbar value
elm_progressbar_value_set(progressbar, 0.6);

// Push the proressbar into the box
elm_box_pack_end(box, progressbar);
evas_object_show(progressbar);

Features

Configuring the Progressbar

To configure the progressbar:

  • Use to progressbar value

    • Change the value with the elm_progressbar_value_set() function. The progressbar emits the changed signal when the progress value changes. In the following example, the progressbar value is set to 20%. 

      /* Supported Style: default */
elm_progressbar_value_set(progressbar, 0.2);

    • Read the current value:

      double value = elm_progressbar_value_get(progressbar);
  • Set a pulse mode

    • Activate the progressbar pulse mode to make the progressbar loop infinitely between the start and end position:

      /* Supported Style: pending_list, process, process/groupindex, process/small, process/popup/small */
elm_progressbar_pulse_set(progressbar, EINA_TRUE);
elm_progressbar_pulse(progressbar, EINA_TRUE);

      if you use pulse true in default style. 'unit' is invisible.

  • Inverted Progressbar

    • Invert the progressbar. In the inverted mode the high values are on the left and the low values on the right.

      /* Supported Style: default */
elm_progressbar_inverted_set(progressbar, EINA_TRUE);
