Check

Check is a UI component which accepts user input by the method of toggling. Check is similar to the radio component, except that it does not work as a group. This document teaches you the basic usage and main features of check component. For more information, see API Reference.

Basic Usage

Take the following steps to use a check in your application:

  1. Add a check with elm_check_add() function. 
    Evas_Object *check;

check = elm_check_add(parent);
  2. Set a style and fill the check with a text or an image according to the style.
    • Set a style to the check with elm_object_style_set() function. You can use the default style without this step. 
      elm_object_style_set(check, "favorite");
    • Set a label to the check with elm_object_text_set() or elm_object_part_text_set(). 
      // default style
elm_object_text_set(check, "Check");

      // on&off style
elm_object_part_text_set(check, "on", "True");
elm_object_part_text_set(check, "off", "False");
    • Set an icon to the button with elm_object_part_content_set() function and pass the part name icon as a parameter. 
      Evas_Object *icon;

elm_object_part_content_set(check, "icon", icon);
  3. Register callback functions. 
    evas_object_smart_callback_add(check, "changed", changed_cb, data);

    void changed_cb(void *data, Evas_Object *obj, void *event_info)
{
   dlog_print(DLOG_INFO, LOG_TAG, "The value has changed\n");
}
Example: the screenshot is after clicking the check. 
Evas_Object *win, *conf, *nf, *box, *check;

// Starting right after the basic EFL UI Layout code
// win - conformant - naviframe

// add a box to pack a check
box = elm_box_add(nf);
elm_object_content_set(nf, box);
evas_object_show(box);
elm_naviframe_item_push(nf, "Check", NULL, NULL, box, NULL);

check = elm_check_add(box);
evas_object_show(check);
elm_box_pack_end(box, check);
