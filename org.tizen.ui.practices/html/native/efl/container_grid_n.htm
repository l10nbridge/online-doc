Grid

Grid is a container component that places its elements at specific positions along a fixed grid, where the position of each object is given as a percentage of the full width and height of the grid. This document explains the basic usage of grid component. For more information, see API Reference.

Figure: The structure of grid component

Basic Usage

Building a layout with a grid usually has the following process:

  1. Add a grid with elm_grid_add() function.
    2. Evas_Object *grid;

grid = elm_grid_add(parent);
  2. Set the size of the grid with elm_grid_size_set() function. The size is 100 x 100 by default.
    3. grid = elm_grid_size_set(grid, 20, 30);
  3. Add objects and pack them into the grid with elm_grid_pack() function.
    4. Evas_Object *button;

elm_grid_pack(grid, button, 0, 0, 10, 10);

The following example shows a simple use case of grid container:

Example: A simple use case of grid container 
Evas_Object *win, *conf, *nf, *grid, *button;

// Starting right after the basic EFL UI layout code
// (win - conformant - naviframe)

// Add a grid to contain buttons and push the grid into the naviframe
grid = elm_grid_add(nf);
elm_object_content_set(nf, grid);
evas_object_show(grid);
elm_naviframe_item_push(nf, "Grid", NULL, NULL, grid, NULL);

// Add a button to the grid
button = elm_button_add(grid);
elm_object_text_set(button, "Button1");
elm_grid_pack(grid, button, 10, 10, 40, 20);
evas_object_show(button);

// Add a button to the grid
button = elm_button_add(grid);
elm_object_text_set(button, "Button2");
elm_grid_pack(grid, button, 50, 50, 40, 30);
evas_object_show(button);

// Add a button to the grid
button = elm_button_add(grid);
elm_object_text_set(button, "Button3");
elm_grid_pack(grid, button, 10, 80, 30, 10);
evas_object_show(button);
