Circle Slider

The circle slider component changes corresponding to rotary events. The circle slider not only extends the UI feature of the slider component (elm_slider), but also replaces the functionalities of the elm_slider in the circular design. For more information, see the Efl Extension Circle Slider API.

Basic Usage

  1. To create a circle slider component, use the eext_circle_object_slider_add() function. Pass a circle surface as the second parameter. 
    Evas_Object *slider = NULL ;

slider = eext_circle_object_slider_add(ad->nf, ad->circle_surface);
  2. Configure the slider. The following configurations are optional. Unless you configure, the slider adopt default styles.
    • Set a range of the slider.
    • Set a color of the slider.

      • You could configure the slider in various ways. For more information, see this section.

  3. Deal with a event
    • Register callback functions 
       evas_object_smart_callback_add(slider, "value,changed", _value_changed_cb, NULL);
    • Implement callback function 
      // Callback for the "value,changed" signal
// Called when the value of the circle slider is changed

static void
_value_changed_cb(void *data, Evas_Object *obj, void *event_info)
{
	double _val = eext_circle_object_value_get(obj) ;
	dlog_print(DLOG_INFO, LOG_TAG, "Circle slider value changed. %0.1f\n", _val);
}

The following example shows a simple use case of circle slider component:

Example: A simple use case of circle slider component 
static void
_value_changed_cb(void *data, Evas_Object *obj, void *event_info)
{
	double _val = eext_circle_object_value_get(obj) ;
	dlog_print(DLOG_INFO, LOG_TAG, "Circle slider value changed. %0.1f\n", _val);
}

Evas_Object *circle_surface, *nf, *slider, *conform ;

// Starting right after the basic EFL UI layout code
// (win - conformant - naviframe)

// Create eext circle surface
circle_surface = eext_circle_surface_conformant_add(conform);
eext_object_event_callback_add(nf, EEXT_CALLBACK_BACK, eext_naviframe_back_cb, NULL);
eext_object_event_callback_add(nf, EEXT_CALLBACK_MORE, eext_naviframe_more_cb, NULL);

slider = eext_circle_object_slider_add(nf, circle_surface);

eext_circle_object_value_min_max_set(slider, 0.0, 10.0);
eext_circle_object_value_set(slider, 5.0);

eext_circle_object_color_set(slider,255,0,0,255);

eext_rotary_object_event_activated_set(slider, EINA_TRUE);
evas_object_smart_callback_add(slider, "value,changed", _value_changed_cb, 0);

Features

  1. Circle slider properties manipulation APIs : Items

    Table: Circle slider properties
    APIs Description

    eext_circle_object_value_set()

    eext_circle_object_value_get()

    		 Set/Get the value of a given circle object.

    eext_circle_object_angle_min_max_set()

    eext_circle_object_angle_min_max_get()

    		 Set/Get the minimum and maximum angle for the item of circle object.

    eext_circle_object_angle_offset_set()

    eext_circle_object_angle_offset_get()

    		 Set/Get the angle offset of a given circle object.

    eext_circle_object_angle_set()

    eext_circle_object_angle_get()

    		 Set/Get the angle in degree of a given circle object.

    eext_circle_object_line_width_set()

    eext_circle_object_line_width_get()

    		 Set/Get the line width of the circle object.

    eext_circle_object_radius_set()

    eext_circle_object_radius_get()

    		 Set/Get circle radius of a given circle object.

    eext_circle_object_color_set()

    eext_circle_object_color_get()

    		 Set/Get the color of circle line and font in a given circle object.

    eext_circle_object_disabled_set()

    eext_circle_object_disabled_get()

    		 Set/Get the disabled state of the circle object.
  2. Circle slider items

    Circle slider consists of two parts, which are "default" and "bg". default item part represents foreground circle ring. and bg part represents background one.

    Table: Circle slider items
    Sample Items

    "default" : Default circle item, which draws a slider bar.

    This item represents the value of the circle slider

    (which is red circular area on the left exmaple)

    "bg" : background circle item.

    (which is dark circular area on the left exmaple)
    • The following examples show how to manipulate items.
      Example: Manipulate circle slider items
      Example code EINA_TRUE 
      Evas_Object* slider;

slider = eext_circle_object_slider_add(ad->nf, ad->circle_surface);

eext_circle_object_value_min_max_set(slider, 0.0, 10.0);
eext_circle_object_value_set(slider, 5.0);

// Set red color
eext_circle_object_color_set(slider,255,0,0,255);

// Set a radius of the foreground circle to 70% by original size.
double default_radius =eext_circle_object_radius_get(slider);
eext_circle_object_radius_set(slider,default_radius*0.7);
      Example: Manipulate circle slider items
      Example code EINA_TRUE 
      Evas_Object* slider;

slider = eext_circle_object_slider_add(ad->nf, ad->circle_surface);

eext_circle_object_value_min_max_set(slider, 0.0, 10.0);
eext_circle_object_value_set(slider, 5.0);
eext_circle_object_color_set(slider,255,0,0,255);

// Set background color
eext_circle_object_item_color_set(slider,"bg",0, 0,255,255);

Callbacks

You can register callback functions connected to the following signals for a slider object:

Table: Circle slider callback signals
Signal Description event_info
value,changed Called when the value of a circle slider is changed. NULL

The signals in API Reference might be more extensive, but only the signals stated above are actually supported by Tizen.
