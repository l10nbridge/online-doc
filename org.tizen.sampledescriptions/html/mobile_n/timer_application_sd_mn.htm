Timer Sample Application

The Timer sample application demonstrates how you can implement a simple time countdown application. The application's user interface consists of a single window, four layouts and a few buttons. The layouts constitute the following UI layers:

  • main application window,
  • custom keypad,
  • custom keypad key,
  • dismiss button.

Figure: Timer application views

This application will notify you when time elapses by playing an alarm sound.

Implementation

Main controller

The time countdown is based on Ecore_Timer. The __timer_cb() callback function is called roughly every 1 second. It is used to decrement the total time that has been initially set. It does so by calling the tm_tick() function from the model module.

static Eina_Bool __timer_cb(void *data)
{
  tm_tick();

  if (tm_get_count() == COUNTDOWN_REACHED) {
    view_show_time_up();
    __start_alarm();
  }

  view_update_time(tm_get_count());

  return ECORE_CALLBACK_RENEW;
}

Another important function is the __ui_cb() callback function. It responds to any UI events, such as:

  • UI_EVENT_START_RESUME - the start or the resume countdown button has been clicked,
  • UI_EVENT_PAUSE - the pause countdown button has been clicked,
  • UI_EVENT_CANCEL - the cancel countdown button has been clicked (while in paused state),
  • UI_EVENT_RESET - the reset button has been clicked (at anytime during the countdown),
  • UI_EVENT_TIME_SET - the countdown time is changed by the user using the keypad.
static void __ui_cb(ui_event_e event)
{
  int hh = 0;
  int mm = 0;
  int ss = 0;
  int count_to_set = 0;

  switch (event) {
  case UI_EVENT_START_RESUME:
    __start_resume_counting();
    break;
  case UI_EVENT_PAUSE:
    __cancel_counting();
    break;
  case UI_EVENT_CANCEL:
    __cancel_counting();
    tm_reset_count();
    __stop_alarm();
    view_update_time(tm_get_count());
    break;
  case UI_EVENT_RESET:
    tm_reset_count();
    view_update_time(tm_get_count());
    __start_resume_counting();
    break;
  case UI_EVENT_TIME_SET:
    view_get_user_time(&hh, &mm, &ss);
    count_to_set = ss + 60 * mm + 60 * 60 * hh;
    tm_set_countdown(count_to_set);
    tm_reset_count();
    view_update_time(tm_get_count());
    break;
  default:
    break;
  }
}

Model

The tm_tick() function which is called every time the timer callback function is invoked updates the current time count. It is as simple as:

void tm_tick(void)
{
  current_sec--;
}
