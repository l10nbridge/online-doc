Updating the Tizen SDK

Using the Tizen Update Manager, you can update the installed Tizen SDK packages to the latest version and install additional packages.

Updating Installed Packages

Update Manager GUI

  1. Close all Tizen SDK programs.

  2. Run the Update Manager:

    • On Mac OS® X, select Launchpad > Update Manager.

    • On Ubuntu, select Dashboard Home > Update Manager.

    • On Windows®, select Start > All Programs > Tizen SDK > Update Manager.

    If you are using Tizen SDK 2.4 rev1 or rev2, the Update Manager Information dialog appears after the Update Manager is automatically updated. Select Update Manager, and click Continue.

    The Update Manager opens:

    The Update Manager finds the packages to update, and the results are displayed on the Installed Packages tab.

  3. To update all packages, click the Update button.

  4. When the packages have been updated, the Update button is disabled and the Updatable icon () disappears.

Note
On Ubuntu, if an authentication dialog appears while updating a package, enter your administrator password.

Update Manager CLI

Run the Update Manager CLI update command using the following syntax:

update-manager-cli update [--accept-license] [--no-java-check] [--proxy <ip:port>] [-f <file full path>] [-p <password>] [<package name>] ... | [-all]

Table: Update command parameters
Parameter Description
-all Updates all available packages.
--accept-license Accepts the license terms.
--no-java-check Skips the Java version check.
-f, --file <file path> If you want to update the SDK from a local SDK image, specify the full path of the SDK image file.
-p, --password <password> Administrator (sudo) password for authentication.

Ubuntu only.
--proxy <value> Proxy configuration value.

Use one of the following values: direct, auto, or ip:port.
<package name> Name of the package you want to update.

You can retrieve the names of updatable packages using the following command:

update-manager-cli show-pkgs

Installing Additional Packages

Update Manager GUI

  1. Run the Update Manager:

    • On Mac OS® X, select Launchpad > Update Manager.
    • On Ubuntu, select Dashboard Home > Update Manager.
    • On Windows®, select Start > All Programs > Tizen SDK > Update Manager.
  2. In the Update Manager, select the All Packages tab.

  3. Click Install next to each package you want to install.

    If you select to install a package that depends on other packages, a confirmation dialog box appears. Click Yes.

Note
On Ubuntu, if an authentication dialog appears while installing a package, enter your administrator password.

Update Manager CLI

Run the Update Manager CLI install command using the following syntax:

update-manager-cli install [--accept-license] [--no-java-check] [--proxy <ip:port>] [-f <file full path>] [-p <password>] [<package name>]

Table: Install command parameters
Parameter Description
--accept-license Accepts the license terms.
--no-java-check Skips the Java version check.
-f, --file <file path> If you want to install packages from a local SDK image, specify the full path of the SDK image file.
-p, --password <password> Administrator (sudo) password for authentication.

Ubuntu only.
--proxy <value> Proxy configuration value.

Use one of the following values: direct, auto, or ip:port.
<package name> Name of the package you want to install.

You can retrieve the names of installable packages using the following command:

update-manager-cli show-pkgs
