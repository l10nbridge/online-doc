Uninstalling the Tizen SDK

You can uninstall individual packages or all packages.

Uninstalling Individual Packages

Update Manager GUI

  1. Run the Update Manager:

    • On Mac OS® X, select Launchpad > Update Manager.

    • On Ubuntu, select Dashboard Home > Update Manager.

    • On Windows®, select Start > All Programs > Tizen SDK > Update Manager.

  2. In the Update Manager, select the Installed Packages tab.

  3. Click Remove next to each package you want to uninstall.

    If you select to uninstall a package that depends on other packages, a confirmation dialog box appears. Click Yes.

Note
On Ubuntu, if an authentication dialog appears while uninstalling a package, enter your administrator password.

Update Manager CLI

Run the Update Manager CLI uninstall command using the following syntax:

update-manager-cli uninstall [-p <password>] [<package name>]

Table: Uninstall command parameters (individual package)
Parameter Description
-p, --password <password> Administrator (sudo) password for authentication.

Ubuntu only.
<package name> Name of the package you want to uninstall.

You can retrieve the names of installed packages using the following command:

update-manager-cli show-pkgs

Uninstalling All Packages

SDK Uninstaller

  1. Run the SDK Uninstaller:

    • On Mac OS® X, select Launchpad > SDK Uninstaller.

    • On Ubuntu, select Dashboard Home > SDK Uninstaller.

    • On Windows®, select Start > All Programs > Tizen SDK > SDK Uninstaller.

    If you are using Tizen SDK 2.4 rev1 or rev2, the Update Manager Information dialog appears after the Update Manager is automatically updated. Select SDK Uninstaller, and click Continue.

    The SDK Uninstaller opens:

  2. Click the Uninstall icon ().

Note
On Ubuntu, if an authentication dialog appears while uninstalling the SDK, enter your administrator password.

Update Manager CLI

Run the Update Manager CLI uninstall command using the following syntax:

update-manager-cli uninstall [-p <password>] [-all]

Table: Uninstall command parameters (all packages)
Parameter Description
-p, --password <password> Administrator (sudo) password for authentication.

Ubuntu only.
-all Uninstalls all packages and removes the Tizen SDK directory.
