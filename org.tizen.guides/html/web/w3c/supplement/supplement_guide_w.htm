Supplementary Features

Supplementary features include extension features that can be used in creating Tizen Web applications.

The main supplementary features are:

  • Typed Array - Khronos

    Enables you to access binary data in JavaScript.

  • WebGL - Khronos

    Enables you to use the WebGL graphics library to create 3D visual elements.

The following guides apply in mobile applications only:

  • FullScreen API - Mozilla

    Enables you to display an element on fullscreen.

The following guides apply in wearable applications only:

  • Camera API (Tizen Extension)

    Enables you to control the camera options, capture images, and record video.
