Web Setting

Tizen enables you to set Web view properties.

This feature is supported in mobile applications only.

The main features of the Web Setting API include:

  • Setting user agents

    You can set a custom user agent string for the running Web application.

  • Deleting cookies

    You can delete all cookies set for the running Web application.
