Bookmark

You can manage the Tizen Web browser bookmarks by retrieving the current bookmark list. You can create or delete both bookmark folders and items.

Note
This feature is supported in mobile applications only.

The main features of the Bookmark API include:

  • Creating bookmarks

    You can create bookmark folders and items using the BookmarkManager interface.

  • Managing bookmarks

    You can retrieve a list of current bookmarks of the Tizen Web browser using the BookmarkManager interface. You can also delete bookmark folders and items.
