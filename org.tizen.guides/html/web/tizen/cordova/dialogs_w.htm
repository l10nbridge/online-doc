Dialogs

Module is for making different types of notifications to the user. The navigator.notification object allows access to the Dialogs API.

The features of the Dialogs API are:

  • Alert

    Show some message (an alert) with one dismiss button with custom text.

  • Confirm

    Show a confirmation dialog box with set of buttons.

  • Prompt

    Show a prompt dialog box where user can put some text, with set of buttons.

  • Beep

    Make a beep sound.

Callbacks

Each time a dialog box is closed, result is provided by calling a callback function

Table: Callback parameters for different dialog box types
Dialog Type Parameters
Alertno parameters
Confirmone parameter: index of pressed button
Promptone paramter: a PromptData object

PromptData contains user input. It has two properties.

PromptData object
Property Value
buttonIndexIndex of a button pressed by user. The index uses one-based indexing, so the values could be 1, 2, 3, etc.
input1Text entered by user.
