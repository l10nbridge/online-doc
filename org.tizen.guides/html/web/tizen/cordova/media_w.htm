Media

Media allows to play and record audio files. Functionalities like play, stop, pause, volume change, and seek to position are presented in tutorial.

Application needs following privilegs:

  • http://tizen.org/privilege/mediacapture privilege to record,
  • http://tizen.org/privilege/volume.set privilege to change volume.
The main features of the Media API are:
  • Play audio file

    You can start audio file playback.

  • Seek to position

    You can change playing position.

  • Change volume

    You can change volume during playback.

  • Get duration and position

    You can get duration of audio file, and retrieve current position.

  • Record audio

    You can start and stop recording.

Playback and status changes

Each time playback status is changed you can be notified by statusCallback parameter for Media constructor. For usage and information on playback status changes see Using status, success, and error callbacks

Example:

var myMedia = new Media(src, successCallback, errorCallback, statusCallback);

Each time status changes, callback will be called with one of the following status constants as an argument.

Table: Playback status changes
Constant Description
Media.MEDIA_NONEfile status is none
Media.MEDIA_STARTINGplayback is starting
Media.MEDIA_RUNNINGplayback is running>
Media.MEDIA_PAUSEDplayback is paused>
Media.MEDIA_STOPPEDplayback is stopped>
