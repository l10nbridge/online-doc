File

The File API gives the possibility to perform operations on files/directories stored in device's filesystem.

To use functionalities based on filesystem, application need to have defined privileges in config.xml file:

  • For reading features:

    <tizen:privilege name="http://tizen.org/privilege/filesystem.read"/>

  • For writing features:

    <tizen:privilege name="http://tizen.org/privilege/filesystem.write"/>
Note
You need to wait for deviceready event before using File Cordova API. 
document.addEventListener( "deviceready", onDeviceReady );

function onDeviceReady() {
    //Cordova is ready...
}
