WidgetService

WidgetService management features include retrieving information about widgets, such as widget name and supported widget size types. You can get the widget instances and receive notifications if the state of the widget changes.

The WidgetService API is mandatory for both Tizen mobile and wearable profiles, which means that it is supported in all mobile and wearable devices. All mandatory APIs are supported on the Tizen Emulators.

The main WidgetService features are:

  • Widget retrieval. Using WidgetServiceManager interface (in mobile and wearable applications) you can:
    • Retrive widget or widgets using getWidgets() method.
    • Receive information about primary widget id or size related to the specific size type.
  • Widget management. Using Widget interface (in mobile and wearable applications) you can:
    • Get name of the widget in given language using getName() method.
    • Get all instances belonging to the widget by getInstances() method.
    • Get variants of specified size type.
    • Receive state change notifications of installed widget.
  • Widget Instance management. Using WidgetInstance interface (in mobile and wearable applications) you can:
    • Change update period of the instance by changeUpdatePeriod() method.
    • Send or get content to the widget instance.
