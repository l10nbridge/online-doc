Tizen
Web Guides

Tizen Web guides introduce the various features that you can use in creating Tizen Web applications.

Note
The API support differs based on the application profile (mobile or wearable). For a complete list of APIs and their supported profiles, see Tizen Web Device API Reference and W3C/HTML5 and Supplementaries API Reference.

Select the feature you are interested in and see what Tizen offers for your application:

  • Guide to Tizen Features

    Enables you to use Tizen features, such as alarms, calendars, and multimedia content, provided by the Tizen Web Device API Reference.

  • Guide to W3C/HTML5 and Some Supplementary Features

    Enables you to use the W3C, HTML5, and some supplementary features provided by the W3C/HTML5 and Supplementaries API Reference.
