Creating Wearable Datetimes

This topic explains how you can create UI components that allow the user to select a date or time.

This feature is supported in wearable applications only.

To create a datetime component:

  • Creating a date picker:

    Create a datetime component and use the elm_object_style_set() function to define the date picker style ("Year : Month : Day").

    The following example creates a date picker and adds it to the conformant component:

    /* Add a datetime component */
datetime = elm_datetime_add(ad->win);
evas_object_size_hint_align_set(datetime, EVAS_HINT_FILL, 0.5);
evas_object_size_hint_weight_set(datetime, EVAS_HINT_EXPAND, 1.0);
/* Set the date picker style */
elm_object_style_set(datetime, "datepicker_layout");
evas_object_show(datetime);

elm_object_content_set(ad->conform, datetime);
  • Creating a time picker:

    Create a datetime component and use the elm_object_style_set() function to define the time picker style ("Hour(12) : Min : AM/PM" or "Hour(24) : Min").

    The following example creates a time picker and adds it to the conformant component:

    /* Add a datetime component */
datetime = elm_datetime_add(ad->win);
evas_object_size_hint_align_set(datetime, EVAS_HINT_FILL, 0.5);
evas_object_size_hint_weight_set(datetime, EVAS_HINT_EXPAND, 1.0);
/* Set the time picker style */
elm_object_style_set(datetime, "timepicker_layout");
evas_object_show(datetime);

elm_object_content_set(ad->conform, datetime);
