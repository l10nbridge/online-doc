Geofence

A geofence is a virtual perimeter for a real-world geographic area. A geofence can be defined by geopoint with radius in case of geopoint geofences, Mac addresses in case of WIFI/Bluetooth geofences.

Geofence feature will alert the user for each and every state change such as IN/OUT.

The main features of Geofence are

  • Enabling the Geofence Manager
  • Defining a geofence
  • Setting the privacy policy
  • Defining the geofence settings

Geofence Manager

The Geofence Manager API is used to get notified about the in/out status depending on the geofence defined by the user

Using GeofenceManager class allows your application to :

  • Create the different types of geofences
  • Get the current state
  • Get in/out alerts

Asynchronous geofence related alerts such as in/out and the event callbacks such as fence added, removed etc are implemented with callback interfaces(functions whose names ends with "cb")

You can use the geofence alerted states as follows

  • If the Geofence Manager is working correctly, the geofence manager is set to GEOFENCE_MANAGER_ERROR_NONE. The device can receive the alerts about the geofence when that particular geofence service is started using geofence_manager_start_geofence API.

  • The alerts are received using three enums namely, GEOFENCE_STATE_UNCERTAIN, GEOFENCE_STATE_IN, GEOFENCE_STATE_OUT in the geofence_state_e structure.

  • If the user revokes a permission to use the location information, the geofence manager state is set to GEOFENCE_MANAGER_PERMISSION_DENIED and the same error is returned back to the application invoking the geofence service

Geofence Definition

Geofence definition is actually defining the parameters of a geofence. There are three types of geofences.

  • Geopoint
  • WIFI
  • Bluetooth

These types can be mentioned while creating the geofences using three enums GEOFENCE_TYPE_GEOPOINT, GEOFENCE_TYPE_WIFI, GEOFENCE_TYPE_BT in the structure geofence_type_e.

  • Geopoint geofences requires a geopoint and a radius.
  • WIFI and Bluetooth geofences requires a mac address

Based on the mentioned type of geofence, Geofence Manager will create the fences for the particular application.
