Screen Mirroring sink

The screen mirroring sink receives shared data from the source device which support Wi-Fi Display and display it.

The screen mirroring sink API gives you the ability to connect to screen mirroring source, start / stop / pause / resume screen mirroring sink, set resolution or display, register callback function to check state change.

You should register the callback function using scmirroring_sink_set_state_changed_cb() to check screen mirroring state. The callback is called for state and error of screen mirroring.

All functions that change the state are synchronous except scmirroring_sink_connect(), scmirroring_sink_start(), scmirroring_sink_pause(), and scmirroring_sink_resume().

Thus the result is passed to the application via the callback mechanism.
