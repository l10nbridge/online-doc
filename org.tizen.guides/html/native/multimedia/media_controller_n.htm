Media Controller

Media controller enables you to communicate between the media controller server and the media controller client.

  • Media controller server

    Enables you to send the server information to the client.

  • Media controller client

    Enables you to request the information to the server.

Media controller server

The media controller server provides the current information about the registered application to the client.

For transfer the information when the client requests, the media controller server updates the state information during the application is active. If application is terminated when the client reqeusts, the media controller server transfer the latest information.

Media controller client

The media controller client can receive the current information from the media controller server via callback.

The client can get the information as following table.

Table: Media controller server state attributes
Attribute Description
MC_SERVER_ACTIVATE The requested media controller server is active
MC_SERVER_DEACTIVATE The requested media controller server is deactive
Table: Media controller server metadata attributes
Attribute Description
MEDIA_TITLE Title of the latest content of the media controller server
MEDIA_ARTIST Artist of the latest content of the media controller server
MEDIA_ALBUM Album name of the latest content of the media controller server
MEDIA_AUTHOR Author of the latest content of the media controller server
MEDIA_GENRE Genre of the latest content of the media controller server
MEDIA_DURATION Duration of the latest content of the media controller server
MEDIA_DATE Date of the latest content of the media controller server
MEDIA_COPYRIGHT Copyright of the latest content of the media controller server
MEDIA_DESCRIPTION Description of the latest content of the media controller server
MEDIA_TRACK_NUM Track number of the latest content of the media controller server
MEDIA_PICTURE Album art of the latest content of the media controller server
Table: Media controller server playback state attributes
Attribute Description
MEDIA_PLAYBACK_STATE_NONE No history of media playback
MEDIA_PLAYBACK_STATE_PLAYING Playback state of playing
MEDIA_PLAYBACK_STATE_PAUSED Playback state of paused
MEDIA_PLAYBACK_STATE_STOPPED Playback state of stopped
MEDIA_PLAYBACK_STATE_NEXT_FILE Playback state of next file
MEDIA_PLAYBACK_STATE_PREV_FILE Playback state of previous file
MEDIA_PLAYBACK_STATE_FAST_FORWARD Playback state of fast forward
MEDIA_PLAYBACK_STATE_REWIND Playback state of rewind
Table: Media controller server shuffle mode state attributes
Attribute Description
SHUFFLE_MODE_ON Shuffle mode is on
SHUFFLE_MODE_OFF Shuffle mode is off
Table: Media controller server repeat mode state attributes
Attribute Description
REPEAT_MODE_ON Repeat mode is on
REPEAT_MODE_OFF Repeat mode is off
