Tizen
Getting Started

The getting started with Tizen documentation provides overall information you need to become familiar with Tizen programming and application model:

  • Preface

    Provides information about the Tizen documentation to help you understand it.

  • Creating Your First Tizen Application

    Demonstrates how you can create a simple HelloWorld application - thus helping you to get more familiar with the Tizen application development process and using the Tizen SDK.

  • Application Development Process

    Teaches you how to create a Tizen application from the initial application planning stage to the end stages where you are ready to run and package your application.

  • Tizen Application Model

    Provides an introduction to the Tizen application life-cycle and package contents.

  • Understanding Tizen Programming

    Introduces the Tizen platform, its architecture, and some useful programming concepts you need to take into account when designing Tizen applications.

  • Tizen Web Sites

    Introduces the various Web sites that you can use in the application development tasks.

For more information about developing Tizen applications, see:

  • Tizen Development Environment and Tools
  • Tizen Tutorials
  • Tizen Guides
  • Tizen API References
