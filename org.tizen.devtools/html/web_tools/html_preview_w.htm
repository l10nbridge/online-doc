HTML Preview

You can preview an HTML file in a browser. Since the preview is based on the Google Chrome™ browser, you have a better rendering result (progressive enhancement) for HTML5 documents than if previewing in the Web Page Editor. All changes made in the HTML code are automatically and immediately reflected in the browser.

The following limitations apply to the HTML preview:

  • The Device API cannot be run in the Emulator.
  • Some network environments do not support live editing.

To preview a HTML file in the HTML Preview view, right-click the project in the Project Explorer view and select Preview, click the preview icon in the toolbar, or use the Ctrl + 4 hotkey. The toolbar icon is shown in the following figure.

Figure: Toolbar icon

Preview acts as your preference as configured by you. You can set the preview in Window > Preferences > Tizen SDK > Web > Live Editing.

Live Highlight

You can synchronize the Editor using Live Highlight. Using this function, the preview shows the cursor's position in the editor in a shadow box.
