UI Builder

The Tizen native UI Builder is a graphical user interface builder tool that simplifies the creation of Tizen native UIs by allowing you to arrange widgets using a drag-and-drop WYSIWYG (What You See Is What You Get) editor.

The UI Builder supports a variety of UI widgets and views and editors.

The UI Builder is not only a UI layout code generating tool, it also supports an easy-to-use programming model though the UI Builder project.

Figure: UI Builder

Supported Views and Editors

The UI Builder supports the following views and editors:

Navigation View

The Navigation view displays thumbnail images of all the views in the currently selected project. If the Navigation view is not visible, open it from Window > Show View > Other > Tizen > Navigation.

Figure: Navigation view

In the Navigation view, you can:

  • Change the current view on the designer by clicking on the thumbnail.
  • Re-order views by dragging them to the selected position.
  • Create and delete view templates and views using the context menu:
    • To add an empty view, right-click the view and select Add Empty View.
    • To add a new view through a view template:

      1. Right-click the view and select New.
      2. Select a category.
      3. Select a view template.
      4. Insert a page ID.
      5. Click Finish.
    • To delete a selected view:
      1. Right-click the view and select Delete.
    • To create a new view template:

      1. Right-click the view and select Create Template.
      2. Input a category.
      3. Insert a view template name.
      4. Click Finish.
    • To set the startup view:
      1. Right-click the view and select Set Startup View.

Outline View

The Outline view displays the hierarchical structure of the page currently open in the Page Designer view. If the Outline view is not visible, open it from Window > Show View > Other > General > Outline.

Figure: Outline view

In the Outline view, you can manage widgets:

  • Change the current view on the designer by selecting the view on the view combobox.
  • Move a widget by dragging it to a new position.
  • Change the widget ID in the context menu or by pressing the F2 key. (The view ID cannot be changed.)
  • Manage selected widgets by cutting, copying, pasting, and deleting them.

Properties View

The Properties view displays the properties of the widget selected in the WYSIWYG Editor view. If the Properties view is not visible, open it from Window > Show View > Other > General > Properties.

The Properties helps you to edit widget and events.

Figure: Properties view

The Properties view consists of Attributes, and Event tabs:

  • Attributes

    In the Attributes tab, you can edit widget attributes. Attributes are categorized into common (common widget properties) and widget specification (widget-specific properties) attributes. The widget specification attributes are divided into categories according to the widget attributes.

    • Common

      The Common category only includes the ID attribute, which is the default widget identifier in the code.

    • Widget specification

      The Widget specification category includes all widget-specific attributes.

  • Event

    In the Event tab, you can create an event handler and bind it. If you click the button, the C editor opens. When the event handler function is focused, you can start writing code.

Resources View

The Resources view displays the resources in the selected project. If the Resources view is not visible, open it from Window > Show View > Other > Tizen > Resources.

Figure: Resources view

The Resources view provides the following features:

  • Resource management

    You can copy, paste, delete, and rename resources.

  • Resource navigation

    You can move files to child and parent folders.

  • Resource filter

    You can filter the view to show resources of a particular name or extension.

  • Resource import

    You can import files or folders.

  • Easy editing

    You can edit the resource property of the widgets using drag and drop.

WYSIWYG Editor

The WYSIWYG Editor view is a WYSIWYG (What You See Is What You Get) editor for the UI Builder project.

Figure: WYSIWYG Editor

The WYSIWYG Editor view consists of the following parts:

  • Palette

    The palette contains EFL widgets, which you can select and add to the design area.

  • Toolbar

    The toolbar supports the following functions:

    • Select the view you want to edit using the view combo.
      View button Description
      Move to the previous view.
      Move to the next view.
    • Select the resolution of the design area using the drop-down menu.
    • Zoom within the design area.
      Zoom button Description
      Zoom out.
      Zoom in.
      Maximize the design area on the screen.
      Change the design area size to a predefined percentage value.

  • Design area

    You can work with widgets visually so that you can see what your application looks like as you build it.
