Table Filtering

The Table Filtering feature of the dynamic analyzer provides only able to show the information corresponding to the keyword set by you in the analysis results table.

The filter properties can be set in the Filter dialog.

Add a filter property and apply it to the table

To use this feature, you must set a filter property in the Filter dialog. To open filter dialog, click dropdown button on the right side of the each analysis results table's title bar, and then select Filter.

Figure: How to open the filter dialog

How to add a filter property is as follows :

  1. Click Add button.
  2. Select a column which you want to set the filter.
  3. Enter a keyword.

Figure: How to add a filter property

Then you click OK, the filter dialog is disappeared and a filter icon is displayed in the upper-right corner of the column set in the filter property.

And the analysis results table's data including the keyword of the filter property appears only.

Figure: Apply the filter property

Verify filter properties

You can verify the filter properties through the filter dialog, and there are two ways to open the filter dialog such as follows :

  1. Click dropdown button on the right side of the each analysis results table's title bar, and then select Filter.

    • You can see all filter properties.

      Figure: Verify filter properties - Filter menu

  2. Click filter icon.

    • You can see all filter properties, and the filter property corresponding to the filter icon you click appears with highlighting in blue color.

      Figure: Verify filter properties - Filter icon

Delete filter properties

How to delete filter properties is to use the filter dialog as well as how to add a filter property.

How to delete a filter property is as follows :

  • Click delete button in the right side of the filter property block.

    Figure: Delete a filter property

When you want to delete all filter properties at once, click Clear button.

Figure: Delete all filter properties
