Uninstalling the Tizen SDK

Uninstalling the Tizen SDK (Since Version 2.4)

You can uninstall individual packages or all packages.

Uninstalling Individual Packages

GUI Uninstallation

  1. Run the Update Manager by clicking the following shortcut:
    • On Ubuntu, go to Dashboard Home > Update Manager.
    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.
    • On Mac OS® X, go to Launchpad > Update Manager.
  2. Click the All Packages tab.
  3. Click the uninstall icon of the package that you want to uninstall.
  4. If you select a package that depends on other packages, a confirmation dialog box appears. Click Yes.
Note
On Ubuntu, if an authentication dialog appears while uninstalling, enter your administrator password.

CLI Uninstallation

  1. Run the CLI Update Manager update command with the following syntax: update-manager-cli uninstall [-p <password>] [<package name>]
Parameters Description
-p, --password <password> Provide the administrator (sudo) password for authentication (Ubuntu only).
<package name> Set the name of the package that you want to uninstall. You can find the installed package name with the command: update-manager-cli show-pkgs

Uninstalling All Packages

GUI Uninstallation

  1. Run the Update Manager by clicking the following shortcut:
    • On Ubuntu, go to Dashboard Home > Update Manager.
    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.
    • On Mac OS® X, go to Launchpad > Update Manager.
  2. Click All Packages tab.
  3. Click Uninstall All.
  4. A confirmation dialog that asks if you want to continue the uninstallation appears. Click Yes.
  5. A confirmation dialog that asks if you want to remove the Tizen SDK directory appears. Click Yes.
Note
On Ubuntu, if an authentication dialog appears while uninstalling, enter your administrator password.

CLI Uninstallation

  1. Run the CLI Update Manager update command with the following syntax: update-manager-cli uninstall [-p <password>] [-all]
Parameters Description
-p, --password <password> Provide the administrator (sudo) password for authentication (Ubuntu only).
-all Uninstall all packages and remove the Tizen SDK directory.

Uninstalling the Tizen SDK (Before Version 2.4)

To remove the Tizen SDK (software development kit) from your computer:

  1. Run the Install Manager of the Tizen SDK.
    • On Ubuntu, go to Applications > Tizen SDK > Install Manager.
    • On Windows®, go to Start > Tizen SDK > Install Manager.
    • On Mac OS® X, go to the tizen-sdk/install-manager directory and run Install Manager.
  2. Check Uninstall the Tizen SDK and click Next.
  3. Click Yes.
  4. After the uninstallation is finished, click Close.
