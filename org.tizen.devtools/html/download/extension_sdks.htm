Extension SDKs

The Tizen SDK supports extension packages, called extra repositories, contributed by external developers and companies.

Extension SDKs (Since Version 2.4)

Adding an Extra Repository

  1. Run the Update Manager by clicking the following shortcut:

    • On Ubuntu, go to Dashboard Home > Update Manager.

    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.

    • On Mac OS® X, go to Launchpad > Update Manager.

  2. Click the All Packages tab.

  3. Click Configuration. The Configuration dialog box appears.

  4. Click the Extra Repository tab.

  5. Click Add. The Add repository dialog box appears.

  6. Enter Name.

  7. Enter Repository that indicates the external server address. You can also click the Folder icon on Repository to select a local image file or local server location.

  8. Click OK.

  9. In the Configuration dialog box, check the repository information in the table.

  10. Click Confirm.

Note
Adding an alternative for the existing repository is not recommended. If you add one anyway, you MUST first deactivate the existing repository and remove all the extra packages from it.

Removing an Extra Repository

  1. Run the Update Manager by clicking the following shortcut:

    • On Ubuntu, go to Dashboard Home > Update Manager.

    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.

    • On Mac OS® X, go to Launchpad > Update Manager.

  2. Click the All Packages tab.

  3. Click Configuration. The Configuration dialog box appears.

  4. Click the Extra Repository tab.

  5. In the table, click the Trash icon for the repository.

  6. Click Confirm.

Note
If the external repository is removed, the packages from the external repository are deprecated.

Modifying an Extra Repository

  1. Run the Update Manager by clicking the following shortcut:

    • On Ubuntu, go to Dashboard Home > Update Manager.

    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.

    • On Mac OS® X, go to Launchpad > Update Manager.

  2. Click the All Packages tab.

  3. Click Configuration. The Configuration dialog box appears.

  4. Click the Extra Repository tab.

  5. In the table, click the target repository and click Modify. The Modify repository dialog box appears.

  6. Make modifications, as needed:

    • Modify Name.

    • Modify Repository that indicates the external server address. You can also click the Folder icon on Repository to select a local image file or local server location.

  7. Click OK.

  8. In the Configuration dialog box, select the check box of the modified repository in the table.

  9. Click Confirm.

Activating or Deactivating an Extra Repository

  1. Run the Update Manager by clicking the following shortcut:

    • On Ubuntu, go to Dashboard Home > Update Manager.

    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.

    • On Mac OS® X, go to Launchpad > Update Manager.

  2. Click the All Packages tab.

  3. Click Configuration. The Configuration dialog box appears.

  4. Click the Extra Repository tab.

  5. In the table, select the check box of the repository to be activated, or clear the check box of the repository to be deactivated.

  6. Click Confirm.

Note
If the external repository is deactivated, the packages from the external repository are not shown on the All Packages tab.

Extension SDKs (Before Version 2.4)

Adding Repositories

To add extra repositories, fetch them from an external server:

  1. Run the SDK Install Manager.

  2. After the SDK is launched, click Next.

  3. Click Extra Repository on the component selection page.

  4. Click Add on the Extra Repository dialog.

  5. Enter the external server and repository name and click OK.

  6. When you see the repository information, click OK.

    The extra packages are displayed below the Extras component.

  7. Select the required packages and click Install.

Removing Repositories

To remove extra repositories:

  1. Run the SDK Install Manager.

  2. After the SDK is launched, click Next.

  3. Click Extra Repository on the component selection page.

  4. In the table, select the repository that you want to remove and click Remove. If the installed packages belong to the selected repository, the packages are removed.
