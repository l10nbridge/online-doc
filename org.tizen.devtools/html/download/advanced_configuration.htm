Advanced Configuration

This page provides advanced configuration of SDK installation for Tizen 2.4 SDK and earlier versions. Please check your SDK version.

Advanced Configuration (Since Version 2.4)

The Update Manager configuration consists of the install option, network proxy setting, and extension extra repository.

Setting the Install Option

You can install or update the Tizen SDK by using one of the following methods. If you change a package repository, installed packages are uninstalled and new packages are installed from the changed repository.

Configuring the Package Server

To configure the Package Server in the Install Option tab:

  1. Run the Update Manager by clicking the following shortcut:
    • On Ubuntu, go to Dashboard Home > Update Manager.
    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.
    • On Mac OS® X, go to Launchpad > Update Manager.
  2. Click the All Packages tab.
  3. Click Configuration. The Configuration dialog box appears.
  4. Click the Install Option tab (selected by default).
  5. In the combo box at the top, select the Package Server.
  6. In the text box, type an available repository address.
  7. Click Refresh.
  8. In the Distribution box, select the distribution you want to download.
  9. Click Confirm.
Note
If the Auto update is selected, the Update Manager automatically updates the latest version of the Tizen SDK.

Configuring the SDK Image

To configure the SDK Image in the Install Option tab:

  1. Run the Update Manager by clicking the following shortcut:
    • On Ubuntu, go to Dashboard Home > Update Manager.
    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.
    • On Mac OS® X, go to Launchpad > Update Manager.
  2. Click the All Packages tab.
  3. Click Configuration. The Configuration dialog box appears.
  4. Click the Install Option tab (selected by default).
  5. In the combo box at the top, select the SDK Image.
  6. Click the folder, browse to the SDK image file, and click OK.
  7. Click Confirm.

Configuring the Proxy

You can configure a proxy to connect to a repository server:

  1. Run the Update Manager by clicking the following shortcut:
    • On Ubuntu, go to Dashboard Home > Update Manager.
    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.
    • On Mac OS® X, go to Launchpad > Update Manager.
  2. Click the All Packages tab.
  3. Click Configuration. The Configuration dialog box appears.
  4. Click the Proxy Settings tab.
  5. Select the configuration you want to use. If you select Manual Proxy Configuration, enter HTTP Proxy and Port.
  6. Click Confirm.

Configuring the Extra Repository

The Tizen SDK supports downloading extension packages from extra repositories which are managed by external developers and companies.

For more information, see Extension SDKs.

Advanced Configuration (Before Version 2.4)

You can choose another version of the Tizen SDK with this functionality. To choose another version:

  1. Run the SDK install manager. The SDK set-up wizard starts.
  2. Click Advanced.
  3. In the Advanced Configuration window, select the Package Server radio button and uncheck the Auto update check button.
  4. Select the SDK version which you want to install and click OK.
  5. Click Next and follow the normal installation procedure.
Note
If the Auto update button is selected, the Install Manager automatically updates the latest version of the Tizen SDK.

For more information, see Tizen SDK package server.
