Advanced Configuration

The Update Manager provides an advanced configuration for the Tizen SDK installation. The advanced configuration consists of the Install Option, Proxy Settings, and Extra Repository.

Configuring the Install Option

You can install or update the Tizen SDK by using one of the following methods. If you change a package repository, installed packages are uninstalled and new packages are installed from the changed repository.

Setting the Package Server

Figure: Configuring the package server

To set the Package Server option in the Install Option tab:

  1. Run the Update Manager by clicking the following shortcut:
    • On Ubuntu, go to Dashboard Home > Update Manager.
    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.
    • On Mac OS® X, go to Launchpad > Update Manager.
  2. Select the All Packages tab.
  3. Click Configuration.
  4. Select the Install Option tab (selected by default).
  5. In the combo box at the top, select Package Server.
  6. In the combo box below Package Server, select an available repository. (The Tizen SDK is distributed through the Amazon S3. If you select the nearby repository in the Alternative locations, you can speed up the SDK download.)
  7. Click Refresh.
  8. In the Distribution box, select the distribution you want to download.
  9. Click Confirm.
Note
If the Auto update check box is selected, the Update Manager automatically updates the latest version of the Tizen SDK.

Setting the SDK Image

Figure: Configuring the SDK image

To set the SDK Image option in the Install Option tab:

  1. Run the Update Manager by clicking the following shortcut:
    • On Ubuntu, go to Dashboard Home > Update Manager.
    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.
    • On Mac OS® X, go to Launchpad > Update Manager.
  2. Select the All Packages tab.
  3. Click Configuration.
  4. Select the Install Option tab (selected by default).
  5. In the combo box at the top, select SDK Image.
  6. Click the Folder icon button, browse to the SDK image file, and click OK.
  7. Click Confirm.

Configuring the Proxy

Figure: Configuring the proxy

You can configure a proxy to connect to a repository server:

  1. Run the Update Manager by clicking the following shortcut:
    • On Ubuntu, go to Dashboard Home > Update Manager.
    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.
    • On Mac OS® X, go to Launchpad > Update Manager.
  2. Select the All Packages tab.
  3. Click Configuration.
  4. Select the Proxy Settings tab.
  5. Select the configuration you want to use. If you select Manual Proxy Configuration, enter HTTP Proxy and Port values.
  6. Click Confirm.

Configuring the Extra Repository

Figure: Configuring the extra repository

The Tizen SDK supports downloading extension packages from extra repositories, which are managed by external developers and companies.
