Updating the Tizen SDK

Updating the SDK (Since Version 2.4)

You can update the latest version of the installed packages and install various Tizen SDK packages through the Tizen SDK Update Manager, since Tizen 2.4 official SDK.

Updating Packages

GUI Updating

  1. Close all Tizen SDK programs.

  2. Run the Update Manager by clicking the following shortcut:

    • On Ubuntu, go to Dashboard Home > Update Manager.

    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.

    • On Mac OS® X, go to Launchpad > Update Manager.

  3. The Update Manager finds the packages to update and the results are displayed on the Update tab.

  4. Click the update icon of the package that you want to update. (To update all packages, click Update All.)

  5. After the update is finished, click Close.

CLI Updating

CLI Update Manager update command has the following syntax:

update-manager-cli update [--accept-license] [--no-java-check] [--proxy <ip:port>] [-f <file full path>] [-p <password>] [<package name>] ... | [-all]

Table: Update command parameters
Parameters Description

-all

Updates all available packages.

--accept-license

Accepts the license terms.

--no-java-check

Skips java version check.

-f, --file <file path>

If you want to update from the local SDK Image, set full path of the SDK Image file.

-p, --password <password>

Ubuntu only. The administrator (sudo) password for authentication.

--proxy <value>

Set proxy configuration value.

Type one of the following values: direct, auto, or ip:port.

<package name>

Package name that you want to update.

You can find updatable package name using follow syntax:

update-manager-cli show-pkgs

Installing Additional Packages

GUI Additional Packages Installing

  1. Run the Update Manager by clicking the following shortcut:

    • On Ubuntu, go to Dashboard Home > Update Manager.

    • On Windows®, go to Start > All Programs > Tizen SDK > Update Manager.

    • On Mac OS® X, go to Launchpad > Update Manager.

  2. Click the All Packages tab.

  3. Click the install icon of the package that you want to install.

  4. If you select a package that depends on other packages, then a confirmation dialog box appears. Click Yes.

Note
On Ubuntu, If an authentication dialog appears while installing, enter your administrator password.

CLI Additional Packages Installing

CLI Update Manager update command has the following syntax.

update-manager-cli install [--accept-license] [--no-java-check] [--proxy <ip:port>] [-f <file full path>] [-p <password>] [<package name>]

Table: Update command parameters
Parameters Description

--accept-license

Accepts the license terms.

--no-java-check

Skips java version check.

-f, --file <file path>

If you want to install from the local SDK Image, set full path of the SDK Image file.

-p, --password <password>

Ubuntu only. The administrator (sudo) password for authentication.

--proxy <value>

Set proxy configuration value.

Type one of the following values: direct, auto, or ip:port.

<package name>

Package name that you want to install.

You can find installable package name using follow syntax:

update-manager-cli show-pkgs

Updating the SDK (Before Version 2.4)

To update the SDK versions before 2.4 official version:

Updating Over the Network

  1. Close all Tizen SDK processes.

  2. Run the SDK Install Manager installed in your Tizen SDK directory. There is no need to download the Install Manager.

    The Install Manager upgrades itself and the welcome page is displayed.

  3. Select Install or Update the Tizen SDK radio button and click Next.

  4. On the Update page, choose additional packages and click Install. The SDK set-up wizard informs you when the installation is complete. That means that the SDK is fully installed and configured on your computer.

Updating with the SDK Image File

  1. Download the SDK image file.

  2. Download a new version of the Install Manager, if the Install Manager cannot access Internet.

  3. Run the SDK Install Manager.

  4. Click Advanced when the SDK set-up wizard starts.

  5. In the Advanced Configuration window, select the SDK Image radio button.

  6. Click the folder button, browse to the SDK image file, and click OK.

  7. Click Next.

  8. Select Install or Update the Tizen SDK radio button and click Next.

  9. On the Update page, choose additional packages and click Install. The SDK set-up wizard informs you when the installation is complete. That means that the SDK is fully installed and configured on your computer.
